# 销售情况仪表盘

这是一个基于 Excel 构建的销售情况分析项目，用于汇总月度成交数据、沉淀历史记录，并通过仪表盘方式展示销售表现。

## 项目结构

- `销售情况仪表盘.xlsx`：主仪表盘文件
- `数据处理/6月_每日销售成交汇总数据.xlsx`：6 月每日成交明细
- `数据处理/7月_每日销售成交汇总数据.xlsx`：7 月每日成交明细
- `数据处理/历史数据.xlsx`：历史成交数据沉淀
- `数据处理/销售人员表-截止8月1日.xlsx`：销售人员基础信息

当前本地实际主文件名若仍显示为 `每月销售数据监控.xlsx`，表示该文件尚未完成重命名并提交。

## 使用技术

- Excel 数据透视表：用于按月份、人员、成交维度快速聚合数据
- Excel 公式：用于指标计算、字段映射和辅助分析
- 图表与仪表盘布局：用于展示成交趋势、人员表现和整体销售情况
- 多工作表数据组织：用于区分原始数据、历史数据、人员信息和结果展示
- Git 与 GitHub：用于版本管理与项目归档

## 分析思维

- 先整理数据源：将不同月份的每日成交数据统一收集，保证字段口径一致
- 再构建历史数据池：把月度数据沉淀到历史数据中，便于横向对比与持续更新
- 再建立分析维度：围绕时间、销售人员、成交结果等维度拆分数据
- 最后做结果展示：通过仪表盘呈现核心指标、趋势变化和人员差异，支持业务复盘

## 项目目标

- 建立可复用的月度销售分析模板
- 提升销售数据汇总与展示效率
- 为销售管理、过程跟踪和结果复盘提供依据
